1. 제목 :  실내용 식물 관리 앱

2. 기능
 1) 하드웨어
   * 물 공급
      - 물 공급원 : 물탱크 ( 사용자가 임의로 정할 수 있도록)
      - 공급 위치 : 화분 속 흙 내부에서 공급
      _ 추가사항 : 물 펌프를 이용하여 물 공급

   * 전원
      - 하드웨어 자체 전원 조절 가능한 스위치 부착
      - 충전형 배터리를 사용하여 전력공급

   * 센서
      - 온도 센서
      - 흙 습도 센서

   * 부가
     - LED라이트 : 전원 상태 알림 용도
     - 사운드 : 상태 알림 용도

  2) 소프트웨어 - 백앤드/프론트앤드 구분없이
    * 화분/식물 관리
      - 사용자가 공급하는 물의 양, 물을 공급 시간을 정할 수 있음.
      - 사용자는 화분의 온도, 습도를 확인 할 수 있음.
      - 사용자는 모듈의 전원상태, 물 공급 정보를 확인 할 수 있음.
      - 사용자는 등록한 식물에 정보를 확인할 수 있음.

    *) 식물 데이터
      - 식물도감 데이터베이스 생성
      - 웹 참조

  3) 추가사항
      - 화분 크기 미정
      - 아두이노 보드판은 나노로 충분할 듯
